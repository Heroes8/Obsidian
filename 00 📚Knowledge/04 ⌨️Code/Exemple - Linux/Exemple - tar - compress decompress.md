**Subject :** commande et option pour rapidement archiver et désarchiver
**Created at :** *2023-04-13*


```bash
# compress
tar -czvf name-of-archive.tar.gz /path/to/directory-or-file
# decompress
mkdir ~/tmp
tar -xzvf archive.tar.gz -C ~/tmp
```
